DataBase ?? ==> a collection of data 
DBMS --> wrapper used to manage the data collection  (oracle,postrgrelsql,mysql,sqllite);

show database --> to view the database list in the serer.

create database --> to create the database.
use databasename --> to make specific db active 
select database() --> to check which db is active 
drop database DBname --> to delete a database

Table --> is a collection of data stored in a structured format 

DataTypes :: 
Numeric  ::  int,smallint,tinyint,mediumint,bigint,decimal,numeric,float,double,bit
string :: char,varchar,binary,varbinary,blob,tinyblob,mediumblob,longblob,text,tinytext,mediumtext,longtext,enum.
date :: date,datetime,timstamp,time,year

varchar(size) --> will store dynamic size with max size specified.

create table name(name type,name type---);

show tables --> to view the tables in the active database
show tablename 
show columns from tablename 
describe tablename || desc tablename
drop table tablename --> delete the whole table 
insert into table(column1, column2) values (value1,value2);
select * from  tablename --> to fetch all data from table 
insert into tablename(name,age) values ("data2",val1),("data2",2),("dat3",3)

NULL --> means data is unknown
NOT NULL --> constraint to add so the data couldn't be NULL while inserting.
DEFAULT --> Constraint used to add a default value to the column while insert.
note : we can set the value to NULL even it has default without NOT NULL.

Primary key --> unique key and not null constraint
PRIMARY KEY (cat_id) --> to set primary key on existing table
auto_increment --> to auto increment the value of primary key / we can use the auto increment for only one column in a table.

Alias --> temporary name to the table or column for current query
select columnname as aliasname from tablename;
select columnname aliasname from tablename;

Update table_name set column_name = value where condition

source filename.sql --> to execute the sql file in mysql command line

STRING Operations :::
CONCAT(a,b,c) ==> abc
CONCAT_WS('separator',a, b,c) ==> aseparatorbseperatorc

substring Operations ::
substring(string, start, Length); ==> in ssms 
substring(string , from length); ==> in mysql
charindex(char,string) ==> return index of character
replace(string, from, to) ==> replace the from with to in string
reverse(string) ==> reverse the string
len(string) ==> return the length of string without counting the trailing spaces in ssms
datalength(string) ==> return the length of bytes used to store the string in ssms include the trailing spaces
upper(string) ==> convert the string to upper case
lower(string) ==> convert the string to lower case
STUFF('Quadratic', 3, 4, 'What');
Left(string, number) ==> return the leftmost number of characters from string
right(string, number) ==> return the rightmost number of characters from string
replicate(string, number) ==> repeat the string for number times in ssms
trim(string) ==> remove the leading and trailing spaces from string in ssms
trim(both 'char' from string) ==> remove the leading and trailing char from string in mysql
trim(leading 'char' from string) ==> remove the leading char from string in mysql
trim(trailing 'char' from string) ==> remove the trailing char from string in mysql

ucase(string) ==> convert the string to upper case
lcase(string) ==> convert the string to lower case
insert(string, position, length, newstring) ==> replace the string with newstring from position and for length characters
repeat(string, number) ==> repeat the string for number times
