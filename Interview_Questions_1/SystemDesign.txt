RateLimiting ::
-------
Builder.Services.AddRateLimitter(opt => {
opt.addwindowLimiters("fixed", options => {
        limiterOptions.PermitLimit = 5; // 5 requests
        limiterOptions.Window = TimeSpan.FromSeconds(10); // per 10 seconds
        limiterOptions.QueueProcessingOrder = QueueProcessingOrder.OldestFirst;
        limiterOptions.QueueLimit = 2; // queue up to 2 extra requests
}
})

PermitLimit: Max requests per window.
Window: Time period for the limit
QueueLimit: Number of requests that can wait if limit is exceeded.
RequireRateLimiting("fixed"): Apply to specific endpoints.
